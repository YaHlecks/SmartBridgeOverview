Features

One App, Every Voice

Four intelligent features work in concert to enable natural, real-time communication regardless of sensory ability.

Speech to Text

Captures spoken language from the blind user in real time and converts it to readable text, delivering messages instantly to the deaf user's screen.

Text to Speech

Converts incoming text and sign language translations into natural-sounding audio, giving the blind user a full auditory experience of the conversation.

Sign Language Translator

Uses YOLOv8 and TFLite computer vision models to detect and classify hand signs in real time through the device camera with high-accuracy recognition.

Sign → Sentence

Sequences individual sign classifications into grammatically coherent sentences, transforming isolated gestures into full, meaningful communication.

Roboflow API Integration

Leverages Roboflow's cloud inference API as a third model tier for additional sign classification accuracy and extended gesture vocabulary coverage.

Bidirectional Channel

Both users operate simultaneously — the deaf user signs while the blind user speaks — enabling a fluid, natural conversation without turn-based constraints.

How It Works

Communication in Four Steps

Smart Bridge handles the translation layer invisibly, so both users can focus on the conversation.

01
Input Capture

The deaf user performs hand signs in front of the camera. The blind user speaks naturally into the microphone. Both channels are active simultaneously.

02
AI Processing

YOLOv8 and TFLite models classify the hand signs frame-by-frame. The speech engine transcribes audio to text in real time with minimal latency.

03
Translation & Assembly

Individual sign labels are assembled into full sentences. Text is packaged for Text-to-Speech output. Roboflow API supplements classification confidence.

04
Delivery

The deaf user reads translated speech on screen. The blind user hears translated signs spoken aloud. Communication flows both ways, seamlessly.

AI Models

Three Models, One Pipeline

Smart Bridge combines on-device inference and cloud API to maximize sign language recognition accuracy across varied environments.

On-Device · Object Detection
YOLOv8

You Only Look Once v8 serves as the primary real-time hand sign detector. Running locally on the device, it processes camera frames at high frame rates to identify and localize hand positions with bounding-box precision — critical for latency-sensitive communication.

Real-time detection On-device inference Bounding boxes Low latency Flutter compatible
On-Device · Classification
TensorFlow Lite

A lightweight TFLite classification model runs directly on the mobile device without network dependency, making it robust for offline use. It classifies detected hand regions into sign labels, optimized for mobile hardware via quantization and pruning.

Mobile optimized Offline capable Quantized model Sign classification Fast inference
Cloud API · Inference
Roboflow API

Roboflow's hosted inference API acts as a third-layer validator and extends the app's gesture vocabulary beyond on-device capacity. HTTP API calls deliver cloud-powered classification results, augmenting confidence scores and covering edge-case signs.

Cloud inference HTTP REST API Extended vocabulary Confidence scoring Model hosting

Abstract

Research Overview

Smart Bridge addresses a critical gap in accessibility technology: the absence of a unified communication channel between individuals who are blind and individuals who are deaf. Existing assistive technologies serve each group independently but fail to enable direct interaction between them.

This study presents the design, implementation, and evaluation of a mobile application that integrates computer vision, speech processing, and machine learning to enable bidirectional communication in real time. Three model tiers — YOLOv8, TensorFlow Lite, and Roboflow's hosted API — provide layered sign language recognition with redundancy and extended vocabulary.

Text-to-Speech and Speech-to-Text modules handle the acoustic channel, while the Sign-to-Sentence pipeline transforms sequential gesture classifications into coherent natural language. The system was evaluated for recognition accuracy, response latency, and usability across both target user groups.
